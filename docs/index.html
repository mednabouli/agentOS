Features

Everything the others assume you'll wire yourself

AgentOS ships a complete orchestration stack — not a toolkit you assemble.

🧠

Extended thinking — native

Orchestrator and debugger use Claude's extended thinking with configurable token budgets per role. No manual prompt wrangling required.

⚠️

Human-in-the-loop approval gates

The plan phase blocks until you approve or reject — from the CLI, TUI, web dashboard, or desktop app. You stay in control.

💰

Per-agent cost tracking

Every agent run records token usage, latency, and cost. The Tracer aggregates totals per task and exposes them in all four interfaces.

🔀

Custom model routing

Route any agent role to Anthropic, AWS Bedrock, or Azure OpenAI via env vars or a routing.json file. Zero code changes.

📦

Template library

5 built-in workflow templates (Stripe, auth, REST API, React component, bug fix) with {{VARIABLE}} substitution. Install custom templates from files or URLs.

🗄️

Local-first state

SQLite by default — no cloud account needed to start. Drop in Supabase credentials when you're ready for team access and RLS policies.

Quick start

From zero to swarm in 30 seconds

No account required. No config file beyond your API key.

shell

Launch a swarm $ agentos run "Add Stripe payments"   $ # Or use a template $ agentos template use add-payments \     PAYMENT_PROVIDER=Stripe \     FEATURE=subscriptions
